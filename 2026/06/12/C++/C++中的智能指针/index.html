C++中的智能指针

智能指针

std::unique_ptr

独占所有权的智能指针，一个对象在同一时间只能被一个 unique_ptr 管理。

std::unique_ptr 基本用法

std::unique_ptr 中的get()方法

获取内部的原始指针，只是“借用”，不拥有。不要手动 delete。可以通过get获得的指针，修改指针指向空间的内容

std::unique_ptr 中的release()方法

释放所有权，但不释放内存。需要手动释放

std::unique_ptr 中的reset()方法

释放当前对象，绑定新的对象。reset接受的是原始指针

std::unique_ptr 转移所有权 std::move()

std::unique_ptr 不允许拷贝

std::unique_ptr 中的 swap() 方法

swap 互换两个 unique_ptr 的所有权

std::unique_ptr 总结

方法 功能 注意点
get() 获取内部指针 不拥有，不能手动 delete
release() 释放所有权，返回原始指针 需手动释放，否则内存泄漏
reset() 释放当前对象，绑定新对象 安全，可多次调用
swap() 交换两个 unique_ptr 的所有权 等同 std::swap(tmp1, tmp2)
operator* 解引用 访问对象
operator-> 调用成员函数 访问对象成员
operator bool 判断是否为空 if(tmp)
get_deleter() 获取自定义删除器 高级用法，主要用于非 delete 资源

std::shared_ptr

多个指针“共享”同一块内存，最后一个指针释放时，内存才会被销毁。

shared_ptr 内部有：

🧠 控制块（control block）

包含：

引用计数（reference count） 、删除器（deleter）

👉 每复制一个 shared_ptr： 引用计数 +1

👉 每销毁一个： 引用计数 -1 ， 当变成 0 → 释放内存

std::shared_ptr 基本用法

std::shared_ptr 中的get()方法

获取内部的原始指针，只是“借用”，不拥有。不要手动 delete。可以通过get获得的指针，修改指针指向空间的内容  获取裸指针（不改变引用计数）

std::shared_ptr 构造

std::shared_ptr支持拷贝构造和移动构造

std::shared_ptr 中的 use_count() 方法

当前有多少个 shared_ptr 指向该对象, use_count是原子操作，是多线程安全的。

std::shared_ptr 中的 unique() 方法

是否唯一拥有（等价于 use_count() == 1）检查和使用不是原子操作。不建议使用

std::shared_ptr 中的 reset() 方法

释放当前资源（引用计数 -1）, 也可以释放当前对象绑定新的对象 reset接受的是原始指针

std::shared_ptr 中的 swap() 方法

交换两个 std::shared_ptr

std::shared_ptr 自定义删除器

自定义析构时候操作， std::make_shared不支持

std::shared_ptr 类型转换

static_pointer_cast

dynamic_pointer_cast

const_pointer_cast

std::shared_ptr 循环引用、自引用

引用计数机制在“环”里失效，导致对象无法释放（内存泄漏）。

循环引用

自引用

程序中要避免循环引用和自引用

std::weak_ptr

weak_ptr 是一个“不拥有对象”的智能指针，它不会增加引用计数。std::weak_ptr不能直接解引用访问所指向的对象。

指针类型 是否拥有对象 是否影响生命周期
shared_ptr ✅ 是 ✅ 会延长
weak_ptr ❌ 否 ❌ 不影响

基础用法

std::weak_ptr 中的 lock() 方法

尝试获取一个 shared_ptr, 返回有效 shared_ptr, 返回 nullptr

std::weak_ptr 中的 expired() 方法

判断对象是否已经被释放。 等同 wp.expired() == (wp.lock() == nullptr)

std::weak_ptr 中的 use_count() 方法

返回当前 shared_ptr（强引用）数量。 多线程下不可靠（瞬时值）

std::weak_ptr 中的reset() 方法

让 weak_ptr 变为空（不再指向对象）

std::weak_ptr 中的 swap() 方法

交换两个 weak_ptr

总结

From \ To unique_ptr shared_ptr weak_ptr 裸指针
unique_ptr ❌ ✅（move） ❌ ✅（get）
shared_ptr ❌ ✅ ✅ ✅（get）
weak_ptr ❌ ✅（lock） ✅ ❌
裸指针 ✅ ⚠️危险 ❌ ✅
